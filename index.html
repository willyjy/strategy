Dynamic Pricing — Start-to-ML Roadmap

A maturity journey. Start with rules, earn organisational trust, then introduce ML. Most failures come from skipping phases.

Phase 1
Rule-Based
Weeks 1–8
Markdown rules · Clearance automation · Competitor price feed
Phase 2
Elasticity
Months 2–4
OLS regression · A/B price tests · Segment elasticity matrix
Phase 3
ML Optimise
Months 4–9
XGBoost demand model · Price simulator · Human review queue
Phase 4
Autonomous
Month 9+
Contextual bandits · RL agent · Real-time pipeline · Circuit breaker
✓
Start this quarter: Competitor price feed + automated clearance markdown rules. Immediate ROI + generates Phase 2 training data.
✓
Quick win: If stock age >45 days → 15% markdown auto-trigger. Removes margin leakage from manual decisions.
⚠
Do not skip Phase 2. Running ML pricing without elasticity data = optimising blind. Phase 2 is the most underrated step.
✗
No autonomous pricing until a price perception risk framework and rollback mechanism are in place.
